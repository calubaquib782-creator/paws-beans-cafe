Easy Update Guide

This simple version is made for GitHub + Netlify. To change a price or menu item, edit menu-data.js.

  1. Open menu-data.js.
  2. Change the item price, name, category, or image path.
  3. Save the file.
  4. In GitHub Desktop, type summary: Update menu.
  5. Click Commit to main, then Push origin.

To change a photo, replace the matching image inside the images folder using the same filename.
