# Paws & Beans Cafe - Supabase Admin Version

Files included:
- index.html - public website
- admin.html - professional admin dashboard
- supabase-config.js - paste your Supabase URL/key here
- supabase-schema.sql - run this once in Supabase SQL Editor

Setup:
1. Supabase > SQL Editor > New query > paste supabase-schema.sql > Run.
2. Supabase > Authentication > Users > Add user. Use your admin email and password.
3. Supabase > Settings > API Keys > copy Publishable key.
4. Paste it in supabase-config.js.
5. Commit and Push with GitHub Desktop.
6. Open /admin.html and login.
7. Click Import starter menu once.
